Grid

Aquí vemos varias formas de trabajar con grid.

Grid no responsive

Hemos definido el grid-container con 3 columnas y 3 filas (Grid explícito), al poner 9 item y darle a item 1 y 5 mas espacio, automaticamente se nos crean mas filas para adaptar los elementos que sobran (grid implícito). Nosotros podemos definir también el tamaño que tendrán las columnas o fias extra si tuviesen que crearse, con la propiedad grid-auto-row y grid-auto-column, por defecto se creará una fila extra, pero también podemos definir que sea una columna con la propiedad grid-auto-flow:column.

Grid responsive

Ahora trabajamos con las propiedades minmax, auto-fill y auto-fit. Tenemos un grid-container2 que ahora si se adapta a lo que le definamos, con minmax le decimos que ancho min y max queremos, en este caso tiene un min de 100px y un max de 1fr. Con auto-fill le decimos que cree mas columnas con la propiedadades de minmax, y con auto-fit le decimos que el contenido de los 3 item se adapten al ancho total.
